.. _地图更新时间:

地图更新时间
------------------------------------------------------------------------------
各个地图上的Npc和怪物死光了之后需要等上一定时间才能够刷新，这个时间对于不同的地图分别是:

1. :ref:`新索匹格` 镇：6个月
2. :ref:`铁拳城堡` ：24个月
3. :ref:`浓雾岛` ：24个月
4. :ref:`布特雷格湾` ：8个月
5. :ref:`自由天堂` ：24个月
6. :ref:`幽灵沼泽` ：6个月
7. :ref:`冰冻高原` ：24个月
8. :ref:`银湾镇` ：24个月
9. :ref:`布莱克郡` ：24个月
10. :ref:`克里格斯` ：8个月
11. :ref:`魔鬼水域` ：8个月
12. :ref:`飞龙沙漠` ：6个月
13. :ref:`隐士岛` ：6个月
14. :ref:`天堂谷` ：6个月
15. :ref:`甜水镇` ：8个月， :ref:`蜂房` 7天
