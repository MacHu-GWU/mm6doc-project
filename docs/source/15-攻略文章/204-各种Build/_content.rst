各种Build
==============================================================================

3巫师1牧师
------------------------------------------------------------------------------
不作弊的情况下的经典Build, 适合有经验的玩家通关.

- 巫师1: 主要施放 ``飞行奇术``, ``群星坠落``
    - 空气大师
    - 拆卸术
- 巫师2: 主要施放 ``时空之门``, ``时空道标``
    - 流水专家
    - 经商术
- 巫师3: 主要施放 ``天佑一日``, ``神佑一时``
    - 光明专家
    - 黑暗专家
    - 空气大师
- 牧师: 主要施放 ``疗伤圣法``
    - 肢体专家
    - 鉴定术
    - 修补术
