经验与升级的关系
==============================================================================
1-2级需要1k经验, 以后每级升级所需经验加1k. 换言之::

    - 1~2: 1k
    - 2~3: 2k
    - 3~4: 3k
    ...

    - n-1~n: (n-1)k

**所以升到n级所需的经验为**: (1 + (n-1)) * (n-1) / 2 = ``n * (n-1) / 2``.

由于前期需要专家级的几个技能是:

- 空气: ``飞行奇术``
- 流水: ``时空道标``
- 肢体: 各种治疗法术
- 光明: ``天佑一日``, ``神佑一时``
- 暗黑: ``护身一日``
- 鉴定术
- 拆卸术

每个人平均只用学2个技术专家, 所以只需要12点技能点. 升到4级能有足够的技能点.
